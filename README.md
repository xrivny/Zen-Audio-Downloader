# Zen Audio Downloader
A logical tool for high-quality audio extraction from URLs.

## Features
- Downloads audio in MP3 format.
- High-quality bitrate (192kbps).
- Simple command-line interface.

## Setup
1. Install requirements: `pip install yt-dlp`
2. Ensure `ffmpeg` is installed on your system.
3. Run: `python audio_downloader.py`
